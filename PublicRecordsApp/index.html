Why I built it

I built this to solve a real problem for our Community Action Team: making minutes easy for the public to access while giving board members fast context from past meetings.

It is live on blyoregon.com (see the public example above), and after sharing it, another board member asked about using the same approach for their organization. This page is the productized version of that solution.
